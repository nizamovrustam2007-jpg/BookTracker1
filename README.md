# Training Planner (План тренировок)
# Автор Ваше Имя и Фамилия
# ОписаниеПриложение для планирования тренировок с возможностью фильтрации по типу тренировки и дате. Данные сохраняются в формате JSON.
# Примеры использования
1. Введите дату в формате ГГГГ-ММ-ДД.
2. Укажите тип тренировки (например, "Силовая", "Кардио").
3. Укажите длительность тренировки в минутах.
4. Нажмите на кнопку "Добавить тренировку", чтобы сохранить запись.
5. Используйте поле фильтрации для поиска по типу тренировки.
